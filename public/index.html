Mumbo Jumbo


What is Mumbo Jumbo? 🧐


Mumbo Jumbo is a collaborative writing game for stimulating creativity in language or writing classrooms. It is vaguely inspired by surrealist writing techniques.

Here's how it works:


  • 🌈The teacher hosts a game and generates a pin number.
  • 🌈Participants join the game using the provided pin.
  • 🌈Students start writing a story.
  • 🌈The teacher shuffles the students' stories.
  • 🌈Each student receives another student's story and has to continue it.
